Release Notes:

  • Version 0.5: added direct links to UHAUL.COM for each item; updated flex-box displays; added note at top of page about Item Settings.
  • Version 0.4: fixed a display bug when the weekend mileage rate was $1.09
  • Version 0.3: fixed some typos.
  • Version 0.2: This is a beta (testing) release of the ability to set item settings like prices and visibility. It has not been thoroughly tested and should be double-checked as you use it. If you find issues, please let me know on Reddit: /u/uhaulcsr or if you're familiar with GitHub, you can open an issue.
  • An additional reminder that the settings are stored per device using a cookie. You'll have to set the settings once per device, and if you clear cookies, they will reset to pre-defined values that may not match your market.
  • This version is pretty rough around the edges. If you're familiar with coding, please feel free to join development on GitHub.
